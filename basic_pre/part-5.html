Basic Computer Course – Theory Questions & Answers (Q101–Q125)

Q101. What is a spreadsheet?
A spreadsheet is a digital application (like MS Excel) used for organizing, analyzing, and storing data in tabular form.
Q102. What is a database?
A database is an organized collection of structured data, typically stored electronically and accessed by computer systems.
Q103. What is a modem?
A modem (modulator-demodulator) is a device that converts digital data to analog signals and vice versa for data transmission over telephone lines.
Q104. What is a hyperlink?
A hyperlink is a clickable link in a document or web page that leads to another location or resource.
Q105. What is a macro in MS Office?
A macro is a set of instructions that automate repetitive tasks in applications like MS Word or Excel.
Q106. What is an antivirus?
Antivirus is software designed to detect, prevent, and remove malicious software from computers.
Q107. What is a backup?
A backup is a copy of data stored separately to protect against data loss or corruption.
Q108. What is a PDF?
PDF (Portable Document Format) is a file format developed by Adobe for presenting documents consistently across devices.
Q109. What is a search engine?
A search engine is a web-based tool (like Google) that helps users find information on the internet.
Q110. What is a domain name?
A domain name is the address of a website that users type in the browser to access a site (e.g., www.example.com).
Q111. What is a browser?
A browser is a software application used to access and view websites on the internet (e.g., Chrome, Firefox).
Q112. What is a screenshot?
A screenshot is a digital image of the visible items displayed on a computer or mobile screen.
Q113. What is a shortcut key?
A shortcut key is a combination of keys that performs a specific function quickly (e.g., Ctrl+C for copy).
Q114. What is a web page?
A web page is a document on the internet, usually written in HTML, and viewed in a web browser.
Q115. What is an email attachment?
An email attachment is a file sent along with an email message.
Q116. What is a spreadsheet cell?
A cell is the intersection of a row and a column in a spreadsheet where data is entered.
Q117. What is an input device?
An input device is hardware used to provide data and control signals to a computer (e.g., keyboard, mouse).
Q118. What is an output device?
An output device is hardware that receives data from a computer and presents it (e.g., monitor, printer).
Q119. What is a file extension?
A file extension is a suffix at the end of a filename that indicates the file type (e.g., .docx, .jpg).
Q120. What is a folder?
A folder is a virtual container used to organize files on a computer.
Q121. What is a recycle bin?
Recycle Bin is a folder in Windows where deleted files are temporarily stored before permanent removal.
Q122. What is a taskbar?
The taskbar is a bar at the bottom of the Windows desktop that shows open applications and provides quick access to system functions.
Q123. What is a CPU socket?
A CPU socket is a slot on the motherboard where the processor (CPU) is installed.
Q124. What is a web server?
A web server is a computer system that hosts websites and delivers web pages to users over the internet.
Q125. What is a command prompt?
The command prompt is a command-line interface in Windows used to execute text-based commands.
