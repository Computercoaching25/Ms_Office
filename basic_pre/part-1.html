Q1. What is a computer?
A computer is an electronic device that processes data and performs tasks according to instructions given by a program. It can store, retrieve, and process information quickly and accurately.
Q2. What is the difference between data and information?
Data is a collection of raw facts and figures. Information is processed data that is meaningful and useful.
Example: "100" is data; "Marks: 100" is information.
Q3. What are the main components of a computer?
The main components are: Input devices (e.g., keyboard, mouse), Output devices (e.g., monitor, printer), CPU (Central Processing Unit), Memory (RAM, ROM), and Storage devices (e.g., hard disk, pen drive).
Q4. What is hardware and software?
Hardware is the physical part of the computer that you can touch (like monitor, keyboard, CPU).
Software is a set of instructions or programs that run on the hardware (like Windows, MS Word).
Q5. What is the function of the CPU?
The CPU (Central Processing Unit) is the "brain" of the computer. It processes instructions, performs calculations, and manages the flow of information through the computer.[1]
Q6. What are the two main types of computer memory?
Primary memory (RAM and ROM) and Secondary memory (Hard disk, CD, Pen drive, etc.). Primary memory is fast and temporary; secondary memory is permanent and used for storage.[1]
Q7. What is RAM and ROM?
RAM (Random Access Memory) is temporary, volatile memory used to store data and instructions while the computer is running.
ROM (Read Only Memory) is permanent memory that stores important instructions needed to start the computer.[1]
Q8. What is an input device? Give two examples.
An input device is used to enter data and instructions into a computer. Examples: Keyboard and Mouse.[2]
Q9. What is an output device? Give two examples.
An output device displays or prints the results of computer processing. Examples: Monitor and Printer.[2]
Q10. What is the difference between system software and application software?
System software controls and manages the hardware (e.g., Windows, Linux).
Application software is used to perform specific tasks (e.g., MS Word, Excel, Tally).[2]
Q11. What is the full form of CPU, ALU, and RAM?
  • CPU: Central Processing Unit
  • ALU: Arithmetic Logic Unit
  • RAM: Random Access Memory
Q12. What is the function of ALU and Control Unit?
ALU performs arithmetic and logical operations.
Control Unit directs all operations in the computer by sending control signals to other units.[1]
Q13. What is a program?
A program is a set of instructions that tells the computer how to perform a specific task.[2]
Q14. What is a byte, kilobyte, megabyte, gigabyte, and terabyte?
  • 1 byte = 8 bits
  • 1 KB = 1024 bytes
  • 1 MB = 1024 KB
  • 1 GB = 1024 MB
  • 1 TB = 1024 GB
Q15. What is the full form of URL and what does it mean?
URL stands for Uniform Resource Locator. It is the address used to access websites on the Internet (e.g., https://www.google.com).[3]
Q16. What is the Internet?
The Internet is a global network that connects millions of computers worldwide, allowing sharing of information and communication.[3]
Q17. What is an operating system? Give two examples.
An operating system is system software that manages computer hardware and software resources. Examples: Windows and Linux.[1]
Q18. What is a supercomputer? Give one example.
A supercomputer is the most powerful type of computer, used for complex scientific calculations. Example: PARAM (India).[2]
Q19. What is the difference between primary and secondary memory?
Primary memory (RAM, ROM) is fast but temporary and limited in size.
Secondary memory (hard disk, CD, pen drive) is slower but permanent and larger in size.[1]
Q20. What is the function of a printer?
A printer is an output device that produces a hard copy (printed paper) of documents, images, or graphics stored on a computer.[2]
Q21. What is a database?
A database is an organized collection of data that can be easily accessed, managed, and updated.[2]
Q22. What is a virus in computers?
A computer virus is a malicious program that can infect and damage computer files and software.[3]
Q23. What is a file and a folder?
A file is a collection of data or information stored on a computer. A folder is used to organize and store multiple files together.[1]
Q24. What is the full form of HTML?
HTML stands for HyperText Markup Language. It is used to create web pages.[3]
Q25. What is the function of a modem?
A modem is a device that converts digital signals to analog and vice versa, allowing computers to connect to the Internet over telephone lines.[3]
