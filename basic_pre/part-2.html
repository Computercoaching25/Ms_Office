Basic Computer Course – Theory Questions & Answers (Q26–Q50)

Q26. What is a spreadsheet?
A spreadsheet is an application software used for organizing, analyzing, and storing data in tabular form. Example: Microsoft Excel.
Q27. What is the use of a mouse?
A mouse is an input device used to control the pointer on the screen, select objects, open files, and perform drag-and-drop operations.
Q28. What is a web browser? Give two examples.
A web browser is a software application used to access and view websites on the Internet. Examples: Google Chrome and Mozilla Firefox.
Q29. What do you mean by file extension? Give two examples.
A file extension is a suffix at the end of a file name that indicates its type. Examples: .docx (Word document), .jpg (image file).
Q30. What is a shortcut key? Give two examples.
A shortcut key is a key combination that performs a specific function quickly. Examples: Ctrl+C (copy), Ctrl+V (paste).
Q31. What is e-mail?
E-mail (electronic mail) is a method of exchanging digital messages over the Internet.
Q32. What is the recycle bin?
Recycle Bin is a folder on the desktop in Windows OS where deleted files are temporarily stored. You can restore or permanently delete files from here.
Q33. What is formatting in MS Word?
Formatting in MS Word means changing the appearance of text or document, such as font style, size, color, alignment, and spacing.
Q34. What is a search engine? Give two examples.
A search engine is a website that helps users find information on the Internet. Examples: Google and Bing.
Q35. What is a password? Why is it important?
A password is a secret code used to protect accounts and data. It prevents unauthorized access to your computer or online services.
Q36. What is a hyperlink?
A hyperlink is a clickable text or image in a document or web page that takes you to another location, web page, or file.
Q37. What is the use of the Print Preview option?
Print Preview allows you to see how a document will look when printed, so you can make adjustments before printing.
Q38. What is the function of the Save and Save As options?
Save updates the current document with recent changes. Save As lets you save a copy of the document with a new name or location.
Q39. What is a template in MS Word?
A template is a pre-designed document layout that you can use to quickly create new documents with a consistent style and format.
Q40. What is an antivirus?
Antivirus is a software program designed to detect, prevent, and remove viruses and other malicious software from your computer.
Q41. What is the difference between cut, copy, and paste?
Cut: Removes selected text/data and places it on the clipboard.
Copy: Copies selected text/data to the clipboard.
Paste: Inserts the clipboard content at the selected location.
Q42. What is a worksheet in Excel?
A worksheet is a single page in an Excel file (workbook) where you can enter and calculate data in rows and columns.
Q43. What is the difference between Save and Save As?
Save updates the existing file. Save As creates a new file with a different name, type, or location.
Q44. What is a cell in Excel?
A cell is the intersection of a row and a column in a worksheet. Each cell has a unique address (e.g., A1, B2).
Q45. What is the use of the SUM function in Excel?
The SUM function adds up a range of numbers in Excel. Example: =SUM(A1:A5) adds values from A1 to A5.
Q46. What is a slide in MS PowerPoint?
A slide is a single page of a PowerPoint presentation, containing text, images, charts, and other objects.
Q47. What is the function of the Undo and Redo commands?
Undo: Reverses the last action.
Redo: Re-applies an action that was undone.
Q48. What is a database management system (DBMS)?
A DBMS is software that helps create, manage, and organize databases. Examples: MS Access, MySQL, Oracle.
Q49. What is the use of the Zoom feature in MS Word?
The Zoom feature changes the magnification of the document on the screen, making it easier to read or edit.
Q50. What is cloud computing?
Cloud computing is the delivery of computing services like storage, processing, and software over the Internet, allowing access from anywhere.
