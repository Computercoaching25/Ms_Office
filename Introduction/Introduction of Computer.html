Definition:
A computer is an electronic device that processes data according to a set of instructions (called programs). It can perform arithmetic and logical operations at high speed and accuracy, and can store, retrieve, and process data to generate meaningful information.

Key Functions of a Computer

  • Input: Accepts raw data and instructions from input devices (like keyboard, mouse, etc.).
  • Storage: Saves data and instructions for current or future use in memory units.
  • Processing: Executes instructions to manipulate data and produce results.
  • Control: Manages and controls all internal operations and the flow of data.
  • Output: Presents the processed information (results) through output devices (like monitor, printer, etc.).

Important Computer Terms

  • Data: Raw facts and figures that are yet to be processed (e.g., numbers, text, images).
  • Information: Processed and organized data that is meaningful and useful for decision-making.
  • Instruction: A command or order that tells the computer what operation to perform.
  • Processing: The sequence of actions the computer takes to convert data into information.

Hardware and Software

  • Hardware: The physical components of a computer system (e.g., CPU, monitor, keyboard, mouse, hard disk, etc.).
  • Software: A collection of programs and instructions that tell the hardware what to do. Software can be system software (like Windows, Linux) or application software (like MS Word, Photoshop).
