Snipping Tool

  • Snipping Tool is a Windows utility for capturing screenshots (snips) of any area of your screen. You can annotate, save, or share your snips easily.
  • It supports several snip types: Free-form Snip, Rectangular Snip, Window Snip, and Full-screen Snip.
  • After capturing, you can highlight, write, or draw on your snip before saving or sharing.

How to Open Snipping Tool

  1. Click Start, type snipping tool, and select it from the results.
  2. Or press Windows logo key + Shift + S to launch the snipping bar instantly.
  3. You can also use the Print Screen (PrtSc) key on some devices.

Snip Types

  • Free-form Snip: Draw a free-form shape around an object.
  • Rectangular Snip: Drag the cursor to form a rectangle.
  • Window Snip: Select a window, such as a dialog box, to capture.
  • Full-screen Snip: Capture the entire screen.

Features

  • Delay capture by 1-5 seconds to open menus before snipping.
  • Annotate, highlight, and draw on snips.
  • Copy to clipboard, save, print, or share your snip.
  • Edit snips in Paint or Paint 3D for more options.

Common Shortcut Keys

  • Windows + Shift + S: Open snipping bar
  • Ctrl + S: Save snip
  • Ctrl + C: Copy snip to clipboard
  • Alt + N: Create a new snip in the same mode
  • Ctrl + P: Print snip
  • Alt + M: Choose a snipping mode
  • Alt + D: Set delay for snip
  • Esc: Cancel snip
