Monitor & Keyboard - Detailed Notes

Understanding the Output and Input Devices of a Computer

What is a Monitor?

A monitor is an electronic visual display device that shows output from a computer. It allows users to view text, images, videos, and graphical data processed by the computer. Monitors are also known as VDU (Visual Display Unit) or screen.

Types of Monitors

  • CRT (Cathode Ray Tube): Bulky, old technology, uses electron beams.
  • LCD (Liquid Crystal Display): Slim, lightweight, uses liquid crystals and backlight.
  • LED (Light Emitting Diode): LCD with LED backlight, brighter and more efficient.
  • OLED (Organic LED): Uses organic compounds, best contrast and color.
  • Touchscreen Monitor: Allows direct interaction by touch.

Key Features of Monitors

  • Screen Size: Measured diagonally in inches (e.g., 24”, 27”).
  • Resolution: Number of pixels (e.g., 1920x1080 Full HD, 4K).
  • Aspect Ratio: Ratio of width to height (e.g., 16:9).
  • Refresh Rate: Number of times the image updates per second (Hz).
  • Response Time: How quickly a pixel changes color (ms).
  • Brightness & Contrast: Determines clarity and vividness.

Uses of Monitor

  • Viewing and interacting with applications and websites.
  • Watching videos, movies, and games.
  • Editing documents, photos, and videos.
  • Video conferencing and browsing the internet.
  • Professional tasks like designing, programming, and trading.

Importance of Monitor

The monitor is the main output device of a computer. It provides a visual interface for users to interact with the system. A good monitor improves productivity, reduces eye strain, and enhances the overall computing experience.

What is a Keyboard?

A keyboard is the main input device of a computer, allowing users to enter text, numbers, and commands. It consists of keys for alphabets, numbers, symbols, and special functions.

Types of Keyboards

  • Standard Keyboard: 104/105 keys, found in desktops.
  • Multimedia Keyboard: Extra keys for volume, play, internet, etc.
  • Wireless Keyboard: Connects via Bluetooth or RF, no wire needed.
  • Virtual Keyboard: On-screen, used in mobiles/tablets.
  • Ergonomic Keyboard: Designed for comfort and less strain.
  • Gaming Keyboard: Mechanical keys, RGB lights, macro keys.

Keyboard Layout & Keys

  • Alphanumeric Keys: A-Z, 0-9
  • Function Keys: F1 to F12 (special tasks)
  • Modifier Keys: Shift, Ctrl, Alt
  • Navigation Keys: Arrow keys, Home, End, Page Up/Down
  • Numeric Keypad: Calculator-style number keys (right side)
  • Special Keys: Enter, Spacebar, Backspace, Delete, Esc, Tab, Caps Lock

Uses of Keyboard

  • Typing text, numbers, and symbols
  • Giving commands to the computer
  • Shortcuts for quick actions (e.g., Ctrl+C, Ctrl+V)
  • Gaming and multimedia control
  • Programming and data entry

Importance of Keyboard

The keyboard is essential for communication with the computer. It provides fast and accurate data entry, control, and navigation, making it a fundamental device for all users.
