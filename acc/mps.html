MS Paint 2007 All Shortcut Keys with Uses
Shortcut Key Use / Description
Ctrl + NCreate a new picture
Ctrl + OOpen an existing picture
Ctrl + SSave changes to a picture
F12Save the picture as a new file
Ctrl + PPrint a picture
Alt + F4Close a picture and its Paint window
Ctrl + ZUndo a change
Ctrl + YRedo a change
Ctrl + ASelect the entire picture
Ctrl + XCut a selection
Ctrl + CCopy a selection to the Clipboard
Ctrl + VPaste a selection from the Clipboard
Right ArrowMove the selection or active shape right by one pixel
Left ArrowMove the selection or active shape left by one pixel
Down ArrowMove the selection or active shape down by one pixel
Up ArrowMove the selection or active shape up by one pixel
EscCancel a selection
DeleteDelete a selection
Ctrl + BBold selected text (Text tool only)
Ctrl + IItalicize selected text (Text tool only)
Ctrl + UUnderline selected text (Text tool only)
Ctrl + Plus (+)Increase the width of a brush, line, or shape outline by one pixel
Ctrl + Minus (-)Decrease the width of a brush, line, or shape outline by one pixel
Ctrl + EOpen the Properties dialog box
Ctrl + WOpen the Resize and Skew dialog box
Ctrl + Page UpZoom in
Ctrl + Page DownZoom out
F11View a picture in full-screen mode
Ctrl + RShow or hide the ruler
Ctrl + GShow or hide gridlines
F10 or AltDisplay keytips
Shift + F10Show the current shortcut menu (context menu)
F1Open Paint Help
Ctrl + F1Expand or collapse the Ribbon
Ctrl + F4Close Paint
Ctrl + drag (after selection)Create a copy of the selection
Shift (while drawing shapes)Draw a perfect circle, square, or straight line
TabMove to next tool
Shift + TabMove to previous tool
Ctrl + InsertCopy
Shift + InsertPaste
Shift + DeleteCut
Ctrl + HomeScroll to the top-left corner of the picture
Ctrl + EndScroll to the bottom-right corner of the picture
