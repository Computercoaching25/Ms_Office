1. Introduction to MS Excel

  • MS Excel is a spreadsheet program by Microsoft used for organizing, analyzing, and visualizing data.
  • It consists of rows (numbered) and columns (lettered), forming cells where data is entered.
  • Common uses: calculations, data analysis, charts, pivot tables, data management, and automation via formulas and macros.
  • Key features: Home, Insert, Page Layout, Formulas, Data, Review, and View tabs on the Ribbon[1][2][3][4][7].

2. Types of Alignment and Line Spacing

Alignments (Left, Center, Right, Justify) control horizontal text alignment. Line spacing controls vertical space between lines (e.g., Single, 1.5, Double).

3. Tables and Creating Ads

Tables help organize data in rows and columns. They're used in resumes, invoices, forms, and advertisements for layout precision.

4. Insert Picture and Editing

Insert images to visually enrich documents. Use features like crop, brightness/contrast, styles, and layout options for customization.

5. Insert Shapes

Add visual interest and illustrations using shapes such as rectangles, arrows, stars, and flowchart symbols.

6. Insert Text Box and Cover Page

Text boxes allow flexible text placement. Cover pages provide a professional front page with title, author, and date.

7. Create Template in MS Word

Templates save layout and design for repeated use (e.g., business letters, invoices), saving time and maintaining consistency.

8. Insert SmartArt, Chart, Hyperlink, Header & Footer

Use SmartArt and Charts for diagrams/data. Hyperlinks link to websites/files. Headers/Footers display info like titles, dates, page numbers.

9. Complete Page Layout Menu

Customize document setup: margins, orientation, size, columns, page breaks, themes, borders, and watermarks.

10. References Menu in MS Word

Useful for academic writing: manage citations, insert bibliographies, footnotes, endnotes, and a table of contents.

11. Insert Caption and Mark Citation

Add captions to images/tables. Mark citations to automatically reference sources and generate bibliographies.

12. Envelopes and Labels in Mailings Menu

Create and print professional envelopes and address labels directly from Word using mailing tools.

13. Use of Mail Merge

Mail Merge automates personalized letters/emails by connecting to databases like Excel to insert individual names/addresses.

14. Spelling, Grammar and Track Changes

Use spell check, grammar tools to correct text. Track Changes lets multiple users edit collaboratively with visible revisions.

15. View Menu in MS Word

Switch views: Read Mode, Web Layout, Print Layout. Access navigation pane, zoom, gridlines, rulers for layout control.
