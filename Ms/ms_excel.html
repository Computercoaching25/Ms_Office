1. Introduction to MS Excel

  • MS Excel is a spreadsheet program by Microsoft used for organizing, analyzing, and visualizing data.
  • It consists of rows (numbered) and columns (lettered), forming cells where data is entered.
  • Common uses: calculations, data analysis, charts, pivot tables, data management, and automation via formulas and macros.
  • Key features: Home, Insert, Page Layout, Formulas, Data, Review, and View tabs on the Ribbon[1][2][3][4][7].

2. If Condition in MS Excel

  • The IF function checks a condition and returns one value if TRUE, another if FALSE.
  • Syntax: =IF(logical_test, value_if_true, value_if_false)
  • Example: =IF(A1>10, "High", "Low")

3. Advanced If Condition in MS Excel

  • Nested IF: Multiple IF statements inside each other for complex logic.
    Example: =IF(A1>80, "A", IF(A1>60, "B", IF(A1>40, "C", "Fail")))
  • Combined with AND, OR for multiple conditions:
    =IF(AND(A1>=50, B1="Pass"), "Eligible", "Not Eligible")

4. Create Table App in MS Excel

  • Select your data range, go to Insert → Table.
  • Tables allow sorting, filtering, and structured references for formulas.
  • Tables auto-format and add features like total rows and banded rows for readability.

5. Conditional Formatting in MS Excel

  • Conditional Formatting highlights cells based on rules (e.g., color cells > 100 in green).
  • Found under Home → Conditional Formatting.
  • Options: Highlight Cell Rules, Top/Bottom Rules, Data Bars, Color Scales, Icon Sets.

6. Conditional Formatting in Detail in MS Excel

  • Custom rules using formulas: =A1>50 to color cells above 50.
  • Applies to ranges, can use multiple rules with priority order.
  • Manage rules: Conditional Formatting → Manage Rules for editing, deleting, or reordering.
  • Useful for dashboards, trend highlighting, error spotting, etc.

7. Format Option in Home Menu in MS Excel

  • Formatting options: font, size, color, bold, italics, underline, cell fill color, borders, alignment, number formats, cell styles, and more.
  • Found in the Home tab for quick access to common formatting tools[1].

8. Pivot Table in MS Excel

  • Pivot Tables summarize and analyze large datasets easily.
  • Create via Insert → PivotTable, select data range.
  • Drag fields into Rows, Columns, Values, and Filters to rearrange and analyze data dynamically.

9. Page Layout in MS Excel

  • Set up print area, orientation (portrait/landscape), margins, size, background, and themes.
  • Found in Page Layout tab for preparing sheets for printing or presentation[1].

10. Basic to Advanced Formulas in Excel

  • Basic: SUM, AVERAGE, MIN, MAX, COUNT
  • Intermediate: IF, SUMIF, COUNTIF, VLOOKUP, HLOOKUP
  • Advanced: INDEX, MATCH, OFFSET, INDIRECT, ARRAY FORMULAS

11. VLOOKUP, HLOOKUP, and LOOKUP Formulas in Excel

Function Purpose Syntax
VLOOKUP Vertical lookup in a table =VLOOKUP(lookup_value, table_array, col_index, [range_lookup])
HLOOKUP Horizontal lookup in a table =HLOOKUP(lookup_value, table_array, row_index, [range_lookup])
LOOKUP General lookup (vertical/horizontal) =LOOKUP(lookup_value, lookup_vector, [result_vector])

12. Create Invoice/Bill in Excel

  • Design table with columns: Item, Description, Quantity, Unit Price, Total.
  • Use formulas: =Quantity*Unit Price for line totals, =SUM(Total Range) for grand total.
  • Add company logo, customer details, and formatting for professional look.

13. Picture Lookup in Excel

  • Use Camera Tool or Linked Pictures to display images based on cell values.
  • Combine with INDEX/MATCH to change pictures dynamically (requires named ranges and some setup).

14. Define Name and Name Manager in Excel

  • Define Name: Assign a name to a cell or range for easy reference (Formulas → Define Name).
  • Name Manager: View, edit, or delete named ranges (Formulas → Name Manager).
  • Names make formulas more readable and manageable.

15. Age Calculator in Excel

  • Use =DATEDIF(Birthdate, TODAY(), "Y") to calculate age in years.
  • For full details: =DATEDIF(A1, TODAY(), "Y") & " Years, " & DATEDIF(A1, TODAY(), "YM") & " Months"

16. Formula Auditing in Excel

  • Tools for tracing and debugging formulas: Trace Precedents, Trace Dependents, Error Checking.
  • Found in Formulas tab → Formula Auditing group.
  • Helps find errors and understand formula logic in complex sheets.

17. Data Menu in Excel

  • Data tab includes: Sort, Filter, Data Tools (Remove Duplicates, Data Validation), Get External Data, Group/Ungroup, What-If Analysis, Consolidate, and more[1].

18. Consolidate, What-If Analysis, and Data Validation in Excel

  • Consolidate: Combine data from multiple ranges/sheets.
  • What-If Analysis: Tools like Scenario Manager, Goal Seek, Data Tables to test different outcomes.
  • Data Validation: Restrict data entry (e.g., only numbers, dates, dropdown lists).

19. Calculation of EMI using PMT, IPMT, PPMT Functions in Excel

  • PMT: Calculates loan EMI: =PMT(rate, nper, pv)
  • IPMT: Interest portion for a period: =IPMT(rate, per, nper, pv)
  • PPMT: Principal portion for a period: =PPMT(rate, per, nper, pv)
  • Example: =PMT(10%/12, 60, 500000) (for 5-year loan, 10% annual rate, ₹5,00,000 principal)

20. Review Menu in Excel

  • Spell Check, Comments, Notes, Protect Sheet/Workbook, Track Changes.
  • Helps in proofreading, collaboration, and security of your Excel files[1].
