Q31. If 'WATER' is coded as 'XBUFS', how is 'EARTH' coded?
Q31. यदि 'WATER' को 'XBUFS' कोड किया गया है, तो 'EARTH' को कैसे कोड करेंगे?
  1. FBSUI
  2. FBSUI
  3. FBSUJ
  4. FBSUK
Solution / समाधान:
Each letter is replaced by the next letter: E→F, A→B, R→S, T→U, H→I.
So, FBSUI.
हर अक्षर के बाद वाला अक्षर लिया गया है:
उत्तर: FBSUI
Q32. Find the odd one out: 8, 27, 64, 100, 125, 216
Q32. निम्न में से कौन सा अलग है: 8, 27, 64, 100, 125, 216
  1. 64
  2. 100
  3. 125
  4. 216
Solution / समाधान:
All except 100 are perfect cubes.
So, 100 is odd one out.
100 को छोड़कर सभी पूर्ण घन (cube) हैं।
उत्तर: 100

Q34. Which number comes next in the series? 13, 17, 23, 31, 41, ?
Q34. श्रंखला में अगली संख्या क्या होगी? 13, 17, 23, 31, 41, ?

Q35. If 2 + 3 = 13, 3 + 4 = 25, 4 + 5 = ?
Q35. यदि 2 + 3 = 13, 3 + 4 = 25, 4 + 5 = ?
  1. 41
  2. 37
  3. 45
  4. 49
Solution / समाधान:
Pattern: (First × Second) + (First + Second)
2×3=6, 6+7=13; 3×4=12, 12+13=25; 4×5=20, 20+21=41
So, 41.
पैटर्न: (पहली × दूसरी) + (पहली + दूसरी)
उत्तर: 41
Q36. Which is the odd one out?

Q37. If 'BLACK' is coded as 'CNBJL', how is 'WHITE' coded?
Q37. यदि 'BLACK' को 'CNBJL' कोड किया गया है, तो 'WHITE' को कैसे कोड करेंगे?

Q38. What comes next in the series? 2, 6, 12, 20, 30, ?
Q38. श्रंखला में अगली संख्या क्या होगी? 2, 6, 12, 20, 30, ?
  1. 42
  2. 44
  3. 46
  4. 48
Solution / समाधान:
Pattern: n² + n
1²+1=2, 2²+2=6, 3²+3=12, 4²+4=20, 5²+5=30, 6²+6=42
So, 42.
पैटर्न: n² + n
उत्तर: 42
Q39. If 7 + 2 = 81, 6 + 3 = 81, 5 + 4 = ?
Q39. यदि 7 + 2 = 81, 6 + 3 = 81, 5 + 4 = ?
  1. 81
  2. 72
  3. 64
  4. 49
Solution / समाधान:
(First + Second)² = 9² = 81 for all.
So, 5+4=9, 9²=81.
So, 81.
(पहली + दूसरी)² = 9² = 81
उत्तर: 81
Q40. Which is the odd one out: Rose, Lily, Lotus, Mango
Q40. इनमें से कौन सा अलग है: Rose, Lily, Lotus, Mango
  1. Rose
  2. Lily
  3. Lotus

Rose, Lily, Lotus are flowers; Mango is a fruit.

Rose, Lily, Lotus फूल हैं, Mango फल है।
