Q11. In a certain code, if FLOWER is written as EKNVDQ, how is GARDEN written?
Q11. एक कोड में FLOWER को EKNVDQ लिखा जाता है, तो GARDEN को कैसे लिखा जाएगा?
  1. FZQCDM
  2. FZQCDL
  3. FZQCDN
  4. FZQCDO
Solution / समाधान:
Each letter is replaced by the previous letter in the alphabet.
G→F, A→Z, R→Q, D→C, E→D, N→M. So, FZQCDM.
हर अक्षर को उसके पिछले अक्षर से बदला गया है:
G→F, A→Z, R→Q, D→C, E→D, N→M. उत्तर: FZQCDM
Q12. Which is the missing number in the series? 4, 9, 19, 39, ?, 159
Q12. श्रंखला में कौन सी संख्या गायब है? 4, 9, 19, 39, ?, 159
  1. 79
  2. 69
  3. 89
  4. 99
Solution / समाधान:
Pattern: ×2+1, ×2+1, ...
4×2+1=9, 9×2+1=19, 19×2+1=39, 39×2+1=79, 79×2+1=159.
So, answer is 79.
पैटर्न: ×2+1
39×2+1=79
उत्तर: 79

Q15. If 8 + 6 = 48, 7 + 5 = 35, then 6 + 4 = ?
Q15. यदि 8 + 6 = 48, 7 + 5 = 35, तो 6 + 4 = ?
  1. 24
  2. 20
  3. 18
  4. 40
Solution / समाधान:
Pattern: (First × Second) = Result.
8×6=48, 7×5=35, 6×4=24.
So, answer is 24.
पैटर्न: (पहली संख्या × दूसरी संख्या) = उत्तर
6×4=24
उत्तर: 24

Q18. If 2, 6, 12, 20, ?, 42
Q18. यदि 2, 6, 12, 20, ?, 42
  1. 30
  2. 28
  3. 32
  4. 36
Solution / समाधान:
Pattern: n^2 + n
1^2+1=2, 2^2+2=6, 3^2+3=12, 4^2+4=20, 5^2+5=30, 6^2+6=42
So, answer is 30.
पैटर्न: n^2 + n
उत्तर: 30

Q20. If 5 + 3 = 28, 6 + 2 = 48, 7 + 4 = ?
Q20. यदि 5 + 3 = 28, 6 + 2 = 48, 7 + 4 = ?
  1. 77
  2. 72
  3. 76
  4. 74
Solution / समाधान:
Pattern: (First × Second) + (First × Second)
5×3=15, 15+13=28; 6×2=12, 12+36=48; 7×4=28, 28+49=77
So, 77
पैटर्न: (पहली × दूसरी) + (पहली × पहली)
7×4=28, 7×7=49, 28+49=77
उत्तर: 77
