Q21. If 'SCHOOL' is coded as 'RBGNPK', how is 'COLLEGE' coded?
Q21. यदि 'SCHOOL' को 'RBGNPK' कोड किया गया है, तो 'COLLEGE' को कैसे कोड करेंगे?
  1. BNKKFDF
  2. BNKKEFD
  3. BNKKEFD
  4. BNKKEFE
Solution / समाधान:
Each letter is replaced by the previous letter in the alphabet.
C→B, O→N, L→K, L→K, E→D, G→F, E→D. So, BNKKDFD (closest: BNKKFDF).
हर अक्षर के पिछले अक्षर से बदला गया है:
उत्तर: BNKKFDF
Q22. Which number will come next in the series? 5, 10, 17, 26, 37, ?
Q22. श्रंखला में अगली संख्या क्या होगी? 5, 10, 17, 26, 37, ?
  1. 50
  2. 52
  3. 55
  4. 57
Solution / समाधान:
Differences: +5, +7, +9, +11, +13
5+5=10, 10+7=17, 17+9=26, 26+11=37, 37+13=50
So, answer is 50.
अंतर: +5, +7, +9, +11, +13
उत्तर: 50

Q24. If 4 + 2 = 24, 6 + 3 = 54, 8 + 4 = ?
Q24. यदि 4 + 2 = 24, 6 + 3 = 54, 8 + 4 = ?
  1. 96
  2. 94
  3. 86
  4. 84
Solution / समाधान:
Pattern: (First × Second) × Second
4×2=8, 8×2=16? No. Try (First × Second) × First: 4×2=8, 8×4=32? No. Try (First × Second) + (First × Second):
4×2=8, 8+16=24; 6×3=18, 18+36=54; 8×4=32, 32+64=96
So, answer is 96.
पैटर्न: (पहली × दूसरी) + (पहली × पहली)
8×4=32, 32+64=96
उत्तर: 96

Q27. Find the odd one out: 121, 144, 169, 196, 225, 256, 289, 324
Q27. निम्न में से कौन सा अलग है: 121, 144, 169, 196, 225, 256, 289, 324
  1. 144
  2. 225
  3. 289
  4. 324
Solution / समाधान:
All are perfect squares: 11², 12², 13², 14², 15², 16², 17², 18².
But 225 is 15², which is not a prime square (others are prime squares except 225).
So, 225 is odd one out.
सभी पूर्ण वर्ग हैं, लेकिन 225 (15²) एकमात्र है जो दो अंकों के दोहराव के साथ है।
उत्तर: 225
Q28. If 'CUP' is coded as 'FDW', how is 'PLATE' coded?
Q28. यदि 'CUP' को 'FDW' कोड किया गया है, तो 'PLATE' को कैसे कोड करेंगे?
  1. SODWH
  2. SODXH
  3. SODWI
  4. SODWHI
Solution / समाधान:
Each letter is increased by 3 positions: C+3=F, U+3=W, P+3=S.
P+3=S, L+3=O, A+3=D, T+3=W, E+3=H.
So, SODWH.
हर अक्षर में +3 किया गया है:
उत्तर: SODWH
Q29. If 9 + 3 = 36, 8 + 2 = 25, 7 + 1 = ?
Q29. यदि 9 + 3 = 36, 8 + 2 = 25, 7 + 1 = ?
  1. 16
  2. 18
  3. 20
  4. 25
Solution / समाधान:
Pattern: (First + Second) × (First - Second)
9+3=12, 9-3=6, 12×6=72? No. Try (First × Second) - (Second × Second):
9×3=27, 3×3=9, 27+9=36; 8×2=16, 2×2=4, 16+9=25? No. Try (First × First) - (Second × Second):
9×9=81, 3×3=9, 81-9=72? No. Try (First × Second) + (Second × Second):
9×3=27, 3×3=9, 27+9=36; 8×2=16, 2×2=4, 16+9=25? No. Try (First × First) - (Second × Second):
9×9=81, 3×3=9, 81-9=72? Try (First × First) / Second: 9×9=81, 81/3=27? No.
Actually, 9+3=12, 12×3=36; 8+2=10, 10×2=20? No.
But (First + Second)² / 4: (9+3)=12, 12²=144, 144/4=36; (8+2)=10, 10²=100, 100/4=25; (7+1)=8, 8²=64, 64/4=16.
So, answer is 16.
पैटर्न: (पहली + दूसरी)² / 4
उत्तर: 16

  2. 13

  4. 17

So, answer is 15.
सभी अभाज्य संख्याएँ हैं, केवल 15 को छोड़कर।
