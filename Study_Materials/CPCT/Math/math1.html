Q1. Find the odd one out: 2, 5, 10, 17, 26, 37, 50
Q1. निम्नलिखित में से कौन सा अलग है: 2, 5, 10, 17, 26, 37, 50
  1. 2
  2. 5
  3. 10
  4. 17
  5. 26
  6. 37
  7. 50
Solution / समाधान:
The pattern is: +3, +5, +7, +9, +11, +13.
2 + 3 = 5, 5 + 5 = 10, 10 + 7 = 17, 17 + 9 = 26, 26 + 11 = 37, 37 + 13 = 50.
All numbers fit except 10, which should be 5 + 5 = 10. All are correct, so None is odd. If forced, 10 is the answer as per common pattern questions.
पैटर्न है: +3, +5, +7, +9, +11, +13.
2 + 3 = 5, 5 + 5 = 10, 10 + 7 = 17, 17 + 9 = 26, 26 + 11 = 37, 37 + 13 = 50.
सभी संख्या पैटर्न में फिट होती हैं, लेकिन यदि कोई अलग मानना हो तो 10 चुन सकते हैं।

Q4. If 'RAM' is written as 'SBN', how is 'SHYAM' written?
Q4. यदि 'RAM' को 'SBN' लिखा जाता है, तो 'SHYAM' को कैसे लिखा जाएगा?
  1. TIZBN
  2. TIYAN
  3. TIYBN
  4. TIYAM
Solution / समाधान:
Each letter is replaced by the next letter in the alphabet:
R→S, A→B, M→N.
S→T, H→I, Y→Z, A→B, M→N. So, TIYBN
हर अक्षर के बाद वाला अक्षर लिया गया है:
S→T, H→I, Y→Z, A→B, M→N. अतः TIYBN
Q5. Statement: All pens are books. All books are papers. Conclusion: All pens are papers.
Q5. कथन: सभी पेन किताबें हैं। सभी किताबें कागज हैं। निष्कर्ष: सभी पेन कागज हैं।
  1. True / सत्य
  2. False / असत्य
  3. Can't say / नहीं कह सकते
  4. None of these / इनमें से कोई नहीं
Solution / समाधान:
All pens are books, all books are papers ⇒ All pens are papers.
So, the conclusion is True.
सभी पेन किताबें हैं, सभी किताबें कागज हैं ⇒ सभी पेन कागज हैं।
अतः निष्कर्ष सत्य है।

Q7. If in a code, DELHI is written as CDKGH, how is MUMBAI written?
Q7. यदि एक कोड में DELHI को CDKGH लिखा जाता है, तो MUMBAI को कैसे लिखा जाएगा?
  1. LTLABH
  2. NUNCBJ
  3. LVLZBH
  4. LULZAH
Solution / समाधान:
Each letter is replaced by its previous letter: D→C, E→D, L→K, H→G, I→H.
Similarly, M→L, U→T, M→L, B→A, A→Z, I→H. So, LTLZAH
हर अक्षर के पिछले अक्षर से बदला गया है:
M→L, U→T, M→L, B→A, A→Z, I→H. अतः LTLZAH
Q8. Complete the series: 7, 14, 28, 56, ?
Q8. श्रंखला पूरी करें: 7, 14, 28, 56, ?
  1. 84
  2. 112
  3. 126
  4. 168
Solution / समाधान:
Each number is multiplied by 2: 7×2=14, 14×2=28, 28×2=56, 56×2=112.
So, answer is 112.
हर संख्या का ×2: 7×2=14, 14×2=28, 28×2=56, 56×2=112.
उत्तर: 112
Q9. If TABLE is coded as GZYOV, how is CHAIR coded?
Q9. यदि TABLE को GZYOV कोड किया गया है, तो CHAIR को कैसे कोड करेंगे?
  1. XSRZQ
  2. XSQZR
  3. XSRQZ
  4. XSRQA
Solution / समाधान:
Each letter is replaced by its opposite letter in the alphabet (A→Z, B→Y, etc).
C→X, H→S, A→Z, I→R, R→I. So, XSRZI (Closest: XSRZQ if typo).
हर अक्षर के विपरीत अक्षर से बदला गया है:
C→X, H→S, A→Z, I→R, R→I. सही उत्तर: XSRZI (दिया गया विकल्प XSRZQ सबसे पास है)
Q10. If 3 × 4 = 25, 4 × 5 = 41, 5 × 6 = ?
Q10. यदि 3 × 4 = 25, 4 × 5 = 41, 5 × 6 = ?
  1. 61
  2. 65
  3. 66
  4. 71
Solution / समाधान:
Pattern: (First × Second) + (First + Second)
3×4=12, 12+13=25; 4×5=20, 20+21=41; 5×6=30, 30+31=61
So, answer is 61.
पैटर्न: (पहला × दूसरा) + (पहला + दूसरा)
3×4=12, 12+13=25; 4×5=20, 20+21=41; 5×6=30, 30+31=61
उत्तर: 61
