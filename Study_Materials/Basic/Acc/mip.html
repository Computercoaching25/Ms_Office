Math Input Panel

  • Math Input Panel is a Windows tool that lets you write mathematical expressions using a mouse, pen, or touchscreen, and then insert them into other programs that support MathML (Mathematical Markup Language).
  • It is useful for creating equations for Word, OneNote, or any application that supports math input.
  • The panel recognizes handwritten math, converts it to a formatted equation, and allows you to insert it directly into your document.

How to Use

  1. Open Math Input Panel from the Start Menu.
  2. Write your mathematical expression using your mouse, stylus, or finger.
  3. Review the recognized equation above the input area.
  4. If needed, use Erase or Select and Correct to fix any mistakes.
  5. Click Insert to send the equation to your open document or program.

Supported Programs

  • Microsoft Word
  • OneNote
  • PowerPoint
  • Some web browsers and other MathML-compatible software

Features

  • Handwriting recognition for math symbols and equations
  • Insert math as MathML into compatible programs
  • Easy correction with erase and select tools
  • Supports fractions, exponents, roots, integrals, summations, and more
