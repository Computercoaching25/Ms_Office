Basic Computer Course – Theory Questions & Answers (Q76–Q100)

Q76. What is a firewall?
A firewall is a security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
Q77. What is cloud storage?
Cloud storage is a service that allows data to be stored on remote servers accessed via the internet, providing scalability and remote access.
Q78. What is the difference between hardware and firmware?
Hardware is the physical components of a computer, while firmware is software permanently programmed into hardware to control its functions.
Q79. What is the purpose of an IP address?
An IP address uniquely identifies a device on a network, allowing it to communicate with other devices over the internet or local networks.
Q80. What is the difference between volatile and non-volatile memory?
Volatile memory (like RAM) loses data when power is off, while non-volatile memory (like ROM, hard drives) retains data without power.
Q81. What is a URL?
URL (Uniform Resource Locator) is the address used to access web pages on the internet.
Q82. What is the function of a router?
A router connects multiple networks and directs data packets between them, enabling internet access and communication.
Q83. What is phishing?
Phishing is a cyber attack where attackers impersonate trustworthy entities to steal sensitive information like passwords and credit card numbers.
Q84. What is a cache memory?
Cache memory is a small, fast memory located close to the CPU that stores frequently accessed data to speed up processing.
Q85. What is the difference between analog and digital data?
Analog data is continuous and varies smoothly, while digital data is discrete and represented in binary form (0s and 1s).
Q86. What is a URL shortener?
A URL shortener is a tool that converts a long URL into a shorter, easier-to-share link.
Q87. What is the primary function of an operating system?
The primary function of an operating system is to manage hardware resources and provide a user interface for interaction with the computer.
Q88. What is the difference between a laptop and a desktop computer?
A laptop is portable with an integrated screen and keyboard, while a desktop is a stationary computer with separate components.
Q89. What is a software bug?
A software bug is an error or flaw in a program that causes it to produce incorrect or unexpected results.
Q90. What is a URL?
URL stands for Uniform Resource Locator; it is the address used to access resources on the Internet.
Q91. What is the function of a compiler?
A compiler translates source code written in a programming language into machine code that a computer can execute.
Q92. What is a cookie in web browsing?
A cookie is a small piece of data stored on a user's computer by a website to remember information about the user.
Q93. What is a VPN?
A VPN (Virtual Private Network) creates a secure connection over the internet, protecting privacy and data.
Q94. What is the function of the Control Panel in Windows?
The Control Panel allows users to view and change system settings and controls in Windows.
Q95. What is the difference between HTTP and HTTPS?
HTTP is the protocol for transferring web pages; HTTPS is the secure version that encrypts data for privacy and security.
Q96. What is a motherboard?
The motherboard is the main circuit board of a computer that connects all components and allows communication between them.
Q97. What is a pixel?
A pixel is the smallest unit of a digital image or display, representing a single point of color.
Q98. What is cloud computing?
Cloud computing provides on-demand access to computing resources like servers and storage over the internet.
Q99. What is the difference between primary and secondary storage?
Primary storage is fast, volatile memory like RAM; secondary storage is slower, non-volatile like hard drives and SSDs.
Q100. What is the function of a graphics card?
A graphics card processes and renders images, videos, and animations for display on the monitor.
