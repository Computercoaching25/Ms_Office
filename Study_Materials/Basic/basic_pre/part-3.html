Basic Computer Course – Theory Questions & Answers (Q51–Q75)

Q51. What does IC stand for in computers?
IC stands for Integrated Circuit. It is a small electronic device made out of a semiconductor material and used in computers and other electronics.[1]
Q52. Which unit sends the Read/Write signal in a computer?
The CPU (Central Processing Unit) sends the Read/Write signal to memory and storage devices.[1]
Q53. Which memory cell requires external circuitry?
Dynamic memory cells (DRAM) require external circuitry for refreshing the data stored in them.[1]
Q54. Which language is used to design web pages?
HTML (HyperText Markup Language) is used to design web pages.[1]
Q55. What does HTML stand for?
HTML stands for HyperText Markup Language.[1]
Q56. What does LAN stand for?
LAN stands for Local Area Network.[1]
Q57. What is a page in PowerPoint called?
A page in PowerPoint is called a Slide.[1]
Q58. Piracy is similar to which term?
Piracy is similar to Copyright violation, or making illegal copies of software.[1]
Q59. What is an Excel sheet?
An Excel sheet is a combination of rows and columns used for organizing and analyzing data (a worksheet).[1]
Q60. What is a handout in PowerPoint?
A handout is a printed copy of a presentation given to the audience for reference.[1]
Q61. What identifies the contents of a field in a database?
The field name identifies the contents of a field in a database.[1]
Q62. What is the product of data processing?
The product of data processing is Information.[1][2]
Q63. What is the process of putting data into a storage location called?
The process of putting data into a storage location is called Writing.[1]
Q64. What is the process of copying data from a memory location called?
The process of copying data from a memory location is called Reading.[1]
Q65. What is a list of instructions used by a computer called?
A list of instructions used by a computer is called a Program.[1][2]
Q66. The CPU consists of which main parts?
The CPU consists of Control Unit, Arithmetic Logic Unit (ALU), and Primary Storage (main memory).[2]
Q67. What is true about primary storage?
Primary storage is part of the CPU, allows very fast access to data, and is relatively more expensive than secondary storage.[2]
Q68. Which is the most powerful type of computer?
The most powerful type of computer is a Supercomputer.[1][7]
Q69. What are software instructions intended to satisfy a user's specific needs called?
They are called Application Software.[1][2]
Q70. What is data redundancy?
Data redundancy means the same data field is stored more than once in a file, leading to duplication.[1]
Q71. What are data security threats?
Data security threats include privacy invasion, hardware failure, and fraudulent manipulation of data.[1]
Q72. What does updating a database mean?
Updating a database means modifying or adding records to the database.[1]
Q73. What is firmware?
Firmware is software that is permanently stored on hardware, usually in ROM chips.[1][2]
Q74. What is the correct sequence the CPU follows for each instruction?
The CPU follows the Fetch - Decode - Execute sequence for each instruction.[1][2]
Q75. What is the ascending order of data hierarchy?
The correct order is: Bit → Byte → Field → Record → File → Database.[1][2]
