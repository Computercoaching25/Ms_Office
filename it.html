What is a Computer?

A computer is an electronic device that processes input data and delivers output results based on instructions given through programs. It is used for calculations, data processing, communication, design, and automation.

Characteristics of a Computer:

  • Speed
  • Accuracy
  • Automation
  • Storage
  • Versatility
  • Diligence

Types of Computers:

  • Supercomputer
  • Mainframe Computer
  • Mini Computer
  • Microcomputer (Personal Computer)

Basic Components of a Computer:

  • Input Unit (Keyboard, Mouse)
  • Central Processing Unit (CPU)
  • Output Unit (Monitor, Printer)
  • Storage Devices (HDD, SSD, Pen Drive)
